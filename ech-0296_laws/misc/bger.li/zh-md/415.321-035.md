# Beschluss des Regierungsrates über die Festsetzung der Kollegiengeldpauschale und Semesterbeitrag an der Universität Zürich
(vom 23. Juni 1993)
Der Regierungsrat beschliesst:
| |
| --- |
| I.2 Die Kollegiengeldpauschale wird wie folgt festgesetzt: für Auditoren: |
| mit 1–2 WochenstundenFr. 100.— |
| jede weitere WochenstundeFr. 50.— |
| bis höchstensFr. 500.— |
| für Studierende3 Fr. 640.— |
| für Doktorandinnen und Doktoranden3 Fr. 140.— |
| II. Der Semesterbeitrag an den ASVZ wird mit Wirkung ab Wintersemester 1993/94 auf festgesetzt.Fr. 18.— |
| Der Semesterbeitrag an die Unfallkasse wird mit Wirkung ab Wintersemester 1993/94 auf reduziert.Fr. 4.— |
| Der Semesterbeitrag zur Finanzierung von studenti - schen Anlässen wird mit Wirkung ab Wintersemester 1993/94 auf festgesetzt.Fr. 2.50 |
